**9. Daily Workflow During Class**



**9.1 Open the Project Folder**

Each time you attend class, first open your terminal and go to the project folder:

cd machine-learning-class



**9.2 Activate the Virtual Environment**

.venv\\Scripts\\activate



**9.3 Start Jupyter Notebook**

jupyter notebook



**9.4 Work on Class Notebooks**

\- Open the notebook provided by the instructor.

\- Run the cells step by step.

\- Save your work regularly.

\- Keep your terminal open while using Jupyter Notebook.



**9.5 End of Class**

When finished, close Jupyter Notebook and deactivate the virtual environment:

deactivate
